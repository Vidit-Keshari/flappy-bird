# Flappy Bird
Another old recognizable game in VGames style.

> This game does not belong to `Vidit: Old Series` and `Vidit: Old Series` has been closed. This is because maintaining 2 series together was not possible by me alone.

## Game Mechanics
- **Controls**: Click anywhere on canvas, press space key or up arrow to jump.
- **Game Over**: Touching a pipe or the ground or moving up out of view is instant loss.
- **Win**: Umm... Actually the game is infinite, winning isn't an option. Either play, or play.
